== Description ==

Want to display your twitter feeds or Tweets on your website or blog or in the sidebar. This plugin uses PHP to make requests to the Twitter REST API. Twitterply is simple, easy and flexible in usability, you can use to display your twitter tweets or anyone else's tweets using a short-code, PHP or via widget in sidebar. This plugin makes use of Twitter REST API by sending authenticated requests.

== Installation ==

1. Upload the full directory into your wp-content/plugins directory.
2. Use your favourite FTP program like CuteFtp, Core FTP, Filezilla.
3. Go to the plugins administration screen in your WordPress Admin and activate the plugin.
4. Go to the left bar and set the options in Announcement Bar Panel. 

Shortcode:
`[twitterply]`

PHP Function:
`<?php if ( function_exists( "twitterply" ) ) { twitterply(); } ?>`

You can also display your tweets using the included widget.
